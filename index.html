Algo de mi trabajo

Generador de contraseñas

Este mini-proyecto es un generador de contraseñas aleatorias cuya lógica se basa en los "arrays" de Javascript y que me ayudó a comprender conceptos como bucles, funciones, parámetros y algunas otras cosas más que se detallan en el repositorio de Github.

Página de error 404

Esta es una página de muestra para el clásico "Error 404: Page not found". Con un diseño que se basa en la tecnología de flexbox para adaptarse a varios tamaños del viewport (responsive design).

Página de ejemplo

Este sitio lo desarrollé con la intención de demostrar mis habilidades front-end. El diseño fué propuesto en uno de los ejercicios del sitio frontendpractice.com, y fué una oportunidad de mejorar algunos conceptos de tecnologías CSS (Como las propiedades heredadas, cajas flexibles y áreas de cuadrícula).
